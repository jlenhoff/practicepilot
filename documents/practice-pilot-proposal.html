04 — HIPAA Compliance

How We Handle Compliance

Every Practice Pilot build starts with a PHI audit — identifying which workflows touch protected health information and which don't. This determines which tools can be used at each step.

Your stack: Google Workspace (Business) — BAA in place ✓


  • All workflows touching PHI will run through BAA-covered tools only
  • Internal alerts containing no PHI may use any tool
  • Zapier and Make.com will not be used — neither offers an accessible BAA for small practices
  • Google Apps Script runs inside your Google Workspace account — automatically covered under your existing BAA
  • A PHI workflow map will be provided at delivery documenting which data touches which tool
